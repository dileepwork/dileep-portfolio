# 🌌 DILEEP Portfolio - Spatial Zoom Experience

A high-end, immersive personal portfolio built with React and Framer Motion, featuring a cinematic Z-axis zoom navigation system. Instead of traditional vertical scrolling, this site transports users through layered scenes in a 3D-like space.

## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| **Spatial Navigation** | Scroll-linked Z-axis zoom transitions between sections. |
| **Cinematic Interactions** | Smooth animations powered by Framer Motion. |
| **Dynamic 3D Background** | Interactive Three.js fiber environment. |
| **Cyber-Luxe Aesthetic** | Dark mode with refined glassmorphism and modern typography. |
| **Responsive Design** | Fully optimized for all screen sizes. |

## 🚀 Getting Started

Follow these steps to get the project up and running on your local machine.

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation & Run

| Step | Command | Description |
| :--- | :--- | :--- |
| **1** | `git clone https://github.com/your-username/dileep-portfolio.git` | Clone the repository |
| **2** | `cd dileep-portfolio` | Navigate to project folder |
| **3** | `npm install` | Install all dependencies |
| **4** | `npm run dev` | Start development server |
| **5** | `npm run build` | Build for production |

## 🎨 Design System
The project uses a custom-built design system defined in `src/index.css`, featuring:
- **Primary Color**: High-contrast blue-violet gradients.
- **Background**: Deep obsidian (`#050505`).
- **Glassmorphism**: Sophisticated backdrop blurs and subtle borders.
- **Typography**: Modern sans-serif hierarchy.
